# Club Chabab Houara  Official Website

Welcome to the official website of **Club Chabab Houara **, an ambitious football club based in Morocco. Recently promoted to the third division, Chabab Houara is on a mission to make a name for itself in the football world.

## About

This website serves as a central hub for the club’s fans and supporters. It features the upcoming match schedules, scores, player profiles, and much more. The website is designed to keep fans engaged and informed about the club's journey, achievements, and future goals.

## Features

- **Home Page**: Quick access to the latest match scores, upcoming games, and club news.
- **Team Page**: Information about the team, player profiles, and stats.
- **Match Updates**: Weekly match scores that are automatically updated, keeping fans in the loop.
- **Responsive Design**: The website is fully responsive and works seamlessly on both desktop and mobile devices.
- **News Section**: Regular updates and announcements about the team’s activities and events.

## Technologies Used

- HTML
- CSS
- JavaScript
